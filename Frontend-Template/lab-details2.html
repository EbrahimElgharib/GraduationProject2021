On-Off Feedback Control The first type of control is simply the On/Off Feedback Control. It is suited to situations where it is only necessary to keep a process variable between two limits. It requires much simpler equipment when implemented in a control situation. When using on/off control, all devices are either fully on or fully off, 100% open or 100% closed. There is no middle ground possible, so oscillation is often a feature of on/off control systems, as the control law is switching between extremes, driving the output between states.
